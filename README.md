# Observability and Monitoring with SpringBoot Grafana Loki

- 1 - change REAL_IP to your machine ip in prometheus.yml
- 2 - docker-compose up
- 3 - start springboot application
- 3 - access grafana 
- 4 - add prometheus url*
- 5 - add loki url*
- 6 - build your dashboards

don't use localhost *
